# Exercise 2.2 – Practice Number Sizes and Ranges

This C# console application displays the **memory size**, **minimum**, and **maximum** values of various numeric data types in C#.

## 🧠 Purpose

The goal of this exercise is to help students understand:
- How much memory different numeric types use
- The range of values each type can store
- How to use generics and reflection in C#

## 📂 Files Included

- `Program.cs` – Main C# code with logic to display type info
- `Exercise02.csproj` – Project configuration file for .NET
